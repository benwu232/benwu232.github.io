    Projects

    TapMagicPlayer

    TapMagic is an experimental technology of using tapping to control hand-held devices.

    TapMagicPlayer is a try of using TapMagic in a real application, which brings a natural and elegant way to control while playing videos.

    I'm sure that TapMagic has a broad range of uses. If you have any idea or suggestion, please let me know.

    Here is the group: https://groups.google.com/forum/?fromgroups#!forum/tapmagic

    Quantum Phantom

    QP is a concept prototype of a new technology, which turns a webcam into a non-touch pointing device that can directly control the objects on screen, and even draw or write on it. It is based on computer vision, machine learning and pattern recognition.

    The project is here: https://github.com/benwu232/quantum_phantom

    AGRE

    AGRE(Accelerometer based Gesture Recognition Engine) is an experimental project, which analyzes the motion data sampled by Wiimote to recognize gestures of digits and some simple symbols.

    The project is here: https://github.com/benwu232/agre
